# SmartAI Enterprise Marketing Website

Open `index.html` in a browser to view the static marketing website.

The site uses the supplied SmartAI screenshots as product proof and presents SmartAI as an enterprise AI operations platform powered by Artificial Intelligence models developed by ITL.

The hero includes the actual SmartAI logo and an AI-inspired visual layer, with a dedicated intelligence-runtime section explaining how SmartAI interprets intent, applies ITL AI models, uses business context, and presents action-ready output.

Included files:

- `index.html` - website page
- `styles.css` - responsive enterprise marketing styling
- `script.js` - product screenshot selector behavior
- `assets/screenshots` - real SmartAI UI screenshots
- `assets/logos` - SmartAI logo assets
- `qa-desktop.png` and `qa-mobile.png` - rendered QA previews
